# Installation

Via npm
```shell
$ npm i icaro -S
```

## Script import

Via `<script>`

```html
<script src='path/to/icaro.js'></script>
```

Via ES2015 modules

```js
import icaro from 'icaro'
```

Via commonjs

```js
const icaro = require('icaro')
```
